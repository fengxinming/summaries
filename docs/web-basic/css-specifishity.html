# CSS优先级

  • 优先级就近原则，同权重情况下样式定义最近者为准
  • 载入样式以最后载入的定位为准

# 优先级

# 相同权重

权限由高到低

  • 元素标签里 (行内样式/内联样式)
  • 写在<style>标签里 (嵌入样式)
  • 写在单独的 CSS 样式表中 (链接样式)
  • 在样式表中链接其他样式表：@import url(css/styles2.css)

# 不同权重

!important > id > class > tag

!important 优先于一切

# 权重计算

选择器的特殊性值表述为4个部分，用0,0,0,0表示

  • 行间样式的特殊性是1,0,0,0
  • ID选择器的特殊性值，加0,1,0,0
  • 类选择器、属性选择器或伪类，加0,0,1,0
  • 元素和伪元素，加0,0,0,1
  • 通配选择器 * 对特殊性没有贡献，即0,0,0,0
  • !important，它没有特殊性值，但它的优先级是最高的，即1,0,0,0,0
