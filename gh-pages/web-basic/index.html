# 在浏览器中输入网址并回车

在浏览器中输入网址查找一般会分为一下几个步骤：

  • DNS域名解析
  • TCP连接
  • 发送HTTP请求
  • 服务器处理请求并返回HTTP报文
  • 浏览器显示页面
  • TCP断开连接
